# translate it

Translate selected text into Zh-cn.

## Usage

- Select the string to be translated
- ctrl + shift + p
- Input `translate it!`
- A new tab will open in browser

## Feature

- 适配驼峰单词

```
getCouponItem => get coupon item => 获取优惠券项目
```
